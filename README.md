# NetDump

## What is?
Is a project with basic functions for network packet capture.

### Command
netdump [--list] [--interface <interface_name>] [--ping <ip>] [--traceroute <ip>] [--wifi_speed]

## Goals
- [x] List Interface of your own system
- [x] Packet Capture
- [ ] Ping
- [ ] Traceroute
- [ ] Wifi Speed
